SVG

3. Oktober 2017

SVG står for Scalable Vector Graphics og er et 2D vektorbasert bildeformat. Det er bygget opp av lesbar XML-kode. Den store fordelen med SVG-grafikk er at den kan skaleres opp uten at kvaliteten blir dårligere, imotsetning til på andre bildeformater som baserer seg på bitmap-grafikk der piksler blir forstørret. Dette er fordi SVG-grafikken blir skapt ved hjelp av matematiske formler, beregninger og figurer. SVG kan vi benytte for å lage logoer, ikoner, animasjoner, illustrasjoner, diagrammer og skjemaer blant annet. Måter vi kan lage SVG-grafikk på er enten å kode den for manuelt eller å benytte et bildebehandlingsprogram som støtter vektorgrafikk. Eksempler er Inkscape og Adobe Illustrator. Når vi lager SVG-grafikk kan vi enten linke den opp i andre elementer eller kode den rett i HTML'en. SVG kan også manipuleres med CSS eller Javascript. Formatet er støttet i HTML5.
